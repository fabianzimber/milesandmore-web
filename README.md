## ✨ Overview

The **Miles & More Web** application is a standalone Next.js (App Router) frontend designed to provide an unparalleled, ultra-cinematic passenger experience. It consumes data directly from the Miles & More backend, delivering interactive seat maps, real-time live map tracking, and beautifully animated 3D airspace views directly to Twitch viewers.

## 🚀 Key Features

- **Cinematic UI/UX:** Glass-morphism, abstract geometric gradients, and smooth scroll mechanics.
- **Real-Time Passenger Dashboard:** Interactive seat selection, leaderboards, and flight telemetry.
- **3D Airspace Scene:** Immersive visual background using Three.js & R3F.
- **NextAuth Integration:** Secure Twitch-based authentication for admins and moderators.

## 💻 Quick Start

### 1. Installation

```bash
npm install
```

### 2. Configuration

Set up your `.env.local` based on the example. Ensure your `NEXT_PUBLIC_BOT_API_URL` points directly to your backend service instance:

```bash
cp .env.example .env.local
```

### 3. Development Server

Run the fully optimized development environment:

```bash
npm run dev
```
